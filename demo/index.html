Turntable

A small frame-flipping image slider. Drag, scroll, autorotate, or wire it to a range input.

Pointer (horizontal)

Drag left ↔ right across the image.

Range input controller

Drag the slider; the frame follows.

Autorotate

Hands off. 120 ms per frame.

Usage

<script src="dist/turntable.iife.min.js"></script>
<script>
  new Turntable("#myTurntable", { axis: "x" });
</script>
